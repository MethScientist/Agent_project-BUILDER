## 🔧 Code Components

### Core Quality Assessment

**File:** `core/quality_assessor.py` (328 lines)

**Purpose:** Semantic code analysis and quality scoring

**Key Methods:**
- `assess_file(file_path) → QualityReport` - Main entry point
- `_assess_python(content)` - Detects Python-specific issues
- `_compute_score(counts, lines)` - Calculates 0-100 score

**Detects:**
- Undefined names (HIGH severity, -20 points)
- Stub functions (HIGH severity, -10 points)
- Bare exceptions (MEDIUM severity, -5 points)
- Unused imports (LOW severity, -2 points)
- Long lines (LOW severity, -2 points)

**Output:** `QualityReport` with score, severity breakdown, recommendations

**Usage:**
```python
from core.quality_assessor import QualityAssessor
qa = QualityAssessor(project_root, project_map)
report = qa.assess_file("path/to/file.py")
print(f"Quality: {report.overall_score}/100")
```

### Training Data Extraction

**File:** `training/build_dataset.py` (155 lines)

**Purpose:** Extract training examples from execution traces

**Key Functions:**
- `extract_from_runtime_trace(trace_path)` - Parse trace data
- `synthesize_training_examples()` - Create labeled examples
- Output format: JSONL (one example per line)

**Input:** `output/runtime_trace.json` (generated by agent)

**Output:** `training/data/dataset.jsonl` (training examples)

**Example Output:**
```json
{
  "step": "step_name",
  "file_path": "/path/to/generated/file.py",
  "quality_score": 85.0,
  "is_acceptable": true,
  "verifier_status": "passed",
  "dataset_split": "train"
}
```

**Usage:**
```bash
python training/build_dataset.py
# Output: 21 examples extracted to training/data/dataset.jsonl
```

### Training & Analysis

**File:** `training/train_adapter.py` (180 lines)

**Purpose:** Analyze or fine-tune model on quality data

**Two Modes:**

**Mode 1: Lightweight (No Heavy Deps)**
```bash
python training/train_adapter.py --lightweight
# Shows: quality distribution, statistics, recommendations
# Time: 30 seconds
# Deps: Python only
```

**Mode 2: Full LoRA Training**
```bash
pip install torch transformers peft datasets
python training/train_adapter.py \
  --data training/data/dataset.jsonl \
  --output_dir training/models/agent_adapter \
  --epochs 3
# Shows: fine-tuned LoRA weights
# Time: 1-2 hours
# Deps: torch, transformers, peft, datasets
```
